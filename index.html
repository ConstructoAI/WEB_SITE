🌟Comment créer ton premier site web avec GitHub et Render

Un guide simple et amusant pour créer ton site web et le publier sur internet !

Temps estimé : 30-45 minutes | Coût : Gratuit ! | Prérequis : Un ordinateur et une adresse email

🎯Qu'est-ce que tu vas apprendre ?

Dans ce guide super facile, tu vas apprendre à créer ton propre site web et à le publier sur internet pour que tout le monde puisse le voir ! C'est comme construire ta propre maison numérique ! 🏠

1 Créer mon compte GitHub 2 Créer mon premier dépôt 3 Écrire mes fichiers web 4 Envoyer sur GitHub 5 Créer mon compte Render 6 Connecter avec GitHub 7 Publier mon site !

💻Un ordinateur

N'importe quel ordinateur fonctionne !

Astuce : Tu peux même le faire sur une tablette !

Pour créer tes comptes

  • Tu dois pouvoir recevoir des emails
  • C'est tout !

Conseil : Utilise une adresse que tu consultes souvent

Ton assistant IA pour t'aider

📝ÉTAPE 1 : Créer ton compte GitHub (Gratuit !)

🤔Qu'est-ce que GitHub ?

GitHub, c'est comme un grand coffre-fort sur internet où tu peux ranger tous les fichiers de ton site web. C'est gratuit et c'est là que tous les programmeurs du monde entier gardent leur code ! 💾

1.1 Aller sur le site GitHub

  1. Ouvre ton navigateur web (Chrome, Firefox, Safari...)
  2. Tape dans la barre d'adresse : github.com
  3. Tu vois un gros bouton vert "Sign up" ? Clique dessus ! 🟢

1.2 Créer ton compte

  1. Choisis ton nom d'utilisateur - Ce sera comme ton surnom sur GitHub ! (ex: "marie-codeuse" ou "tom-websites")
  2. Tape ton adresse email - Celle que tes parents t'ont donnée
  3. Crée un mot de passe - Au moins 8 caractères avec des lettres et des chiffres (ex: "MonSiteWeb123")
  4. Résoudre le puzzle - GitHub va te demander de prouver que tu n'es pas un robot 🤖
  5. Clique sur "Create account"

1.3 Vérifier ton email

  1. Va dans ta boîte email (Gmail, Yahoo, etc.)
  2. Cherche un email de GitHub - Il sera peut-être dans les spams !
  3. Clique sur le lien dans l'email - Cela va confirmer que l'email est bien à toi

✅Bravo ! Tu as réussi l'étape 1

  • Ton compte GitHub est créé 🎉
  • Ton email est vérifié ✓
  • Tu peux te connecter sur github.com ✓

Félicitations ! Tu es maintenant un membre officiel de la communauté GitHub ! 🚀

📁ÉTAPE 2 : Créer ton premier dépôt (repository)

🤔Qu'est-ce qu'un dépôt ?

Un dépôt (ou "repository" en anglais), c'est comme un dossier sur ton ordinateur, mais sur internet ! C'est là que tu vas mettre tous les fichiers de ton site web. 📂

2.1 Créer un nouveau dépôt

  1. Sur la page d'accueil de GitHub, cherche le bouton vert "New" ou "+"
  2. Clique sur "New repository"
  3. Tu vas voir une page avec plein d'options - ne t'inquiète pas, c'est plus simple que ça en a l'air ! 😊

2.2 Configurer ton dépôt

  1. Nom du dépôt : Tape quelque chose comme "mon-premier-site" ou "mon-site-web" (sans espaces !)
  2. Description : Écris "Mon premier site web créé avec Claude Code !" 🎉
  3. Public ou Privé : Choisis "Public" - comme ça tout le monde pourra voir ton super travail !
  4. Coche la case "Add a README file" - C'est comme la page d'accueil de ton dépôt

2.3 Créer le dépôt

  1. Vérifie que tout est correct
  2. Clique sur le gros bouton vert "Create repository"
  3. TADA ! 🎉 Tu viens de créer ton premier dépôt sur GitHub !

💡Conseil important

N'utilise jamais d'espaces dans le nom de ton dépôt ! Utilise plutôt des tirets (-) ou des underscores (_). Par exemple : "mon-site-web" ou "mon_site_web"

✅Super ! Tu as réussi l'étape 2

  • Ton dépôt est créé 📁
  • Il a un nom cool ✓
  • Il est public pour que tout le monde le voie ✓
  • Il a un fichier README ✓

Tu progresses super bien ! Maintenant on va ajouter tes fichiers web ! 🚀

✍️ÉTAPE 3 : Écrire les fichiers de ton site web

🤔Quels fichiers pour un site web ?

Un site web, c'est comme une maison ! Tu as besoin de :

  • index.html - La structure de ta maison (les murs, les pièces...)
  • style.css - La décoration (les couleurs, les meubles...)
  • script.js - Les équipements électriques (optionnel pour commencer)

3.1 Créer le fichier index.html

  1. Dans ton dépôt GitHub, clique sur "Add file" puis "Create new file"
  2. Dans la boîte "Name your file", tape : index.html
  3. Dans la grande boîte de texte, colle ce code magique :
Mon Premier Site Web !

Bienvenue sur mon site ! 🎉

Salut ! Je m'appelle [Ton Prénom] et c'est mon premier site web !

Je l'ai créé avec l'aide de Claude Code 🤖

Mes passe-temps favoris :

  • Jouer aux jeux vidéo 🎮
  • Lire des livres 📚
  • Apprendre à programmer 💻

Merci de visiter mon site ! 😊

3.2 Créer le fichier style.css

  1. Clique sur "Commit new file" en bas de la page
  2. Clique encore sur "Add file" puis "Create new file"
  3. Cette fois, tape : style.css
  4. Colle ce code pour rendre ton site beau :
body { font-family: Arial, sans-serif; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); color: white; padding: 20px; text-align: center; } h1 { font-size: 3em; text-shadow: 2px 2px 4px rgba(0,0,0,0.5); margin-bottom: 20px; } h2 { color: #ffeb3b; font-size: 2em; } p, li { font-size: 1.2em; line-height: 1.6; } ul { text-align: left; max-width: 400px; margin: 0 auto; background: rgba(255,255,255,0.1); padding: 20px; border-radius: 10px; }

💡Personnalise ton site !

N'oublie pas de remplacer "[Ton Prénom]" par ton vrai prénom dans le fichier HTML ! Tu peux aussi changer les passe-temps par les tiens ! 😊

✅Génial ! Tu as réussi l'étape 3

  • Fichier index.html créé 📄
  • Fichier style.css créé 🎨
  • Ton contenu est personnalisé ✓

Ton site prend forme ! Maintenant on va le mettre en ligne ! 🚀

☁️ÉTAPE 4 : Sauvegarder tes fichiers sur GitHub

🤔Qu'est-ce qu'un "commit" ?

Faire un "commit", c'est comme prendre une photo de ton travail et la sauvegarder ! GitHub va garder cette "photo" pour toujours, comme ça tu ne perdras jamais ton travail ! 📸

4.1 Sauvegarder ton fichier HTML

  1. Assure-toi que ton fichier index.html est bien écrit
  2. En bas de la page, tu vois "Commit new file" ?
  3. Dans la boîte "Commit message", écris : "Ajout de ma page d'accueil HTML"
  4. Clique sur le bouton vert "Commit new file"

4.2 Sauvegarder ton fichier CSS

  1. Fais pareil pour ton fichier style.css
  2. Message de commit : "Ajout des styles CSS pour mon site"
  3. Clique sur "Commit new file"

4.3 Vérifier que tout est là

  1. Retourne à la page principale de ton dépôt
  2. Tu devrais voir :
    • 📄 README.md
    • 📄 index.html
    • 📄 style.css
  3. Si tu vois tout ça, c'est parfait ! 🎉

💡Conseil d'expert

Chaque fois que tu modifies quelque chose sur ton site, n'oublie pas de faire un "commit" ! C'est comme sauvegarder ton jeu vidéo - ça évite de perdre ton progrès ! 🎮

✅Excellent ! Tu as réussi l'étape 4

  • Tes fichiers HTML et CSS sont sauvegardés 💾
  • Ils sont visibles dans ton dépôt ✓
  • GitHub garde une trace de tous tes changements ✓

Ton site est prêt ! Maintenant on va le publier sur internet avec Render ! 🌍

🌐ÉTAPE 5 : Créer ton compte Render (Gratuit !)

🤔Qu'est-ce que Render ?

Render, c'est comme un hôtel pour sites web ! Il va prendre ton site web qui est sur GitHub et le mettre sur internet pour que tout le monde puisse le voir. Et en plus, c'est gratuit ! 🏨

5.1 Aller sur le site Render

  1. Ouvre un nouvel onglet dans ton navigateur
  2. Tape dans la barre d'adresse : render.com
  3. Tu vois le site de Render ? C'est parti ! 🚀

5.2 Créer ton compte Render

  1. Clique sur "Get Started for Free" ou "Sign Up"
  2. Choix malin : Clique sur "Sign up with GitHub" - comme ça pas besoin de retaper tes infos ! 🤓
  3. GitHub va te demander si tu authorises Render - clique "Authorize"
  4. Render va peut-être te demander quelques infos supplémentaires - remplis-les

5.3 Vérifier ton compte

  1. Render peut t'envoyer un email de vérification - va le chercher !
  2. Clique sur le lien dans l'email
  3. Retourne sur Render - tu devrais être connecté ! 🎉

💰Pourquoi Render est gratuit ?

  • Render propose un plan gratuit pour les débutants
  • Parfait pour apprendre et montrer tes projets
  • Ton site sera un peu plus lent qu'un site payant, mais ça marche très bien !
  • Tu pourras toujours passer à un plan payant plus tard si tu veux

✅Génial ! Tu as réussi l'étape 5

  • Ton compte Render est créé 🎉
  • Il est connecté à ton GitHub 🔗
  • Tu es prêt à publier ton site ✓

On y est presque ! Maintenant on va connecter ton site GitHub avec Render ! 🚀

🔗ÉTAPE 6 : Connecter ton GitHub avec Render

🤔Comment ça marche ?

Render va regarder dans ton dépôt GitHub, prendre tes fichiers HTML et CSS, et créer un vrai site web avec une vraie adresse internet ! C'est magique ! ✨

6.1 Créer un nouveau service

  1. Sur la page d'accueil de Render, clique sur "New +" en haut à droite
  2. Dans le menu qui s'ouvre, choisis "Static Site"
  3. Render va te montrer tous tes dépôts GitHub - génial ! 🎉

6.2 Choisir ton dépôt

  1. Trouve ton dépôt dans la liste (celui que tu as créé à l'étape 2)
  2. Clique sur "Connect" à côté de ton dépôt
  3. Si tu ne le vois pas, clique sur "Configure account" pour autoriser plus de dépôts

6.3 Configurer ton site

  1. Name : Donne un nom cool à ton site (ex: "mon-premier-site-web")
  2. Branch : Laisse "main" - c'est parfait !
  3. Root Directory : Laisse vide (ton site est à la racine)
  4. Build Command : Laisse vide (pas besoin pour un site HTML simple)
  5. Publish Directory : Tape juste "." (un point)
  6. Clique sur "Create Static Site"

⏰Patience, ça arrive !

Render va prendre quelques minutes pour créer ton site. Tu vas voir plein de messages qui défilent - c'est normal ! Il construit ton site web. 🏗️

✅Super ! Tu as réussi l'étape 6

  • Ton dépôt GitHub est connecté à Render 🔗
  • Render est en train de construire ton site 🏗️
  • Bientôt tu auras une vraie adresse web ! ✓

Le grand moment approche ! Ton site va bientôt être en ligne ! 🚀
